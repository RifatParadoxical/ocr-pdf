Q: A SEQUENCE IS DEFINED BY $A_1 = 1$, $A_{N+1} = A_N + N + 1$.
FIND A CLOSED-FORM EQUATION AND DETERMINE THE VALUE OF $A_{100}$'.

Ans:
$A_{N+1} = A_N + N+1$
$A_1 = 1$
$A_{1+1} = A_1 + 1 + 1 = 1+1+1$
$\Rightarrow A_2 = 3$
$A_{2+1} = A_2 + 2 + 1 = 3 + 2 + 1$
$\Rightarrow A_3 = 6$
$A_3+1 = A_3 + 3 + 1 = 6 + 3 + 1$
$\Rightarrow A_4 = 10$
$A_{4+1} = A_4 + 4 + 1 = 10 + 4 + 1$
$\Rightarrow A_5 = 15$
$A_1, A_2, A_3, A_4, A_5 = 1, \stackrel{2}{)}, \stackrel{3}{)}, \stackrel{4}{)}, \stackrel{5}{)} 3, 6, 10, 15$

SO, EVERYTIME WE GO FOR N, IT'S LIKE, $A_N = 1+2+3+ \cdot \cdot \cdot +(N-1)+N$
SO THAT'S BINOMIAL N th TRIANGULAR FORMULA, TRIANGULAR FORMULA.
$A_N = \frac{N(N+1)}{2}$

THEREFOR, $A_{100} = \frac{100 \times (100+1)}{2} = \frac{100 \times 101}{2} = \frac{10100}{2} = 5050$
